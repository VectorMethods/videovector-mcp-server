# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Standards

Project spaces should be professional, inclusive, and focused on improving the VideoVector MCP server. Examples of expected behavior include:

- Using welcoming and precise language.
- Respecting different experience levels and viewpoints.
- Giving actionable, good-faith feedback.
- Keeping discussions scoped to the issue, pull request, or design decision.

Examples of unacceptable behavior include harassment, discriminatory language, personal attacks, publishing private information, or using project spaces for spam or abuse.

## Enforcement

Report conduct concerns through the security/contact path listed in [SECURITY.md](./SECURITY.md). Maintainers may remove comments, close issues, block users, or take other reasonable action to protect the project community.
